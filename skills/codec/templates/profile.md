# 用户知识背景

```yaml
用户:
  技术背景:
    语言: []
    框架: []
    领域: []
  熟悉程度:
    # 对项目相关领域的熟悉度：初级 / 中级 / 高级
    相关领域: 初级
    OS 内核: 初级
    网络: 初级
  学习目标:
    - <用户想从这个项目了解什么>
	阅读偏好:
    # 偏好的讲解方式：流程导向 / 概念导向 / 代码导向
    类型: 流程导向
  理解跟踪:
    - 日期: <YYYY-MM-DD>
      本轮目标: <用户本轮想了解什么>
      本轮理解: <本轮结束时确认理解了哪些>
      仍有疑问: ["疑问1", "疑问2"]
      覆盖模块: ["模块A"]
      讲解深度校准:
        - <概念>: 需要更多背景 / 深度合适 / 可以更深
```
